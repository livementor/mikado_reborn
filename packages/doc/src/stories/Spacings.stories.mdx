# Spacings

We are using a `8px` based spacing grid, computed by getting `50%` of the html font-base (which is `16px` by default in most cases) - so the real value depends on the user preferences.

The format for spacings is the following : `[property][side]-[size]`.

Example: `pt-1`

<br/>

### There are two types of properties :

- `m` : for margin
- `p` : for padding

### You can decide on which side to apply a property by using :

- `t` : apply on top
- `r` : apply on right
- `b` : apply on bottom
- `l` : apply on left
- `x` : apply on left and right
- `y` : apply on top and bottom

If no side is provided, the property will be applied on all sides.

### The size represents the rem value for the spacing :

- `auto` : auto
- `0` : 0px
- `0.5` : 0.5rem
- `1` : 1rem
- `1.5` : 1.5rem
- `2` : 2rem
- `3` : 3rem
- `4` : 4rem
- `5` : 5rem
- `6` : 6rem
- `7` : 7rem
- `8` : 8rem
- `9` : 9rem
- `10` : 10rem
