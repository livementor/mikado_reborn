# Getting Started

To use Mikado Reborn library, you must wrap your page in the `mkr-app` component.

<Source
  language='html'
  code={`
<mkr-app>
    <!-- your content here -->
</mkr-app>
  `}
/>

This will provide you with the basic styles like colors and spacings.
